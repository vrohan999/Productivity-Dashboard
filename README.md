# Productivity Dashboard

An interactive web application developed as part of **Project 3 – Interactive Web Elements** for the Frontend Development Internship Program.

## Project Overview

Productivity Dashboard is a JavaScript-powered web application designed to demonstrate DOM manipulation, user interaction, and dynamic content updates. The application allows users to manage tasks, switch between light and dark themes, view a live clock, and generate motivational quotes fetched from an external API.

The primary objective of this project was to implement interactive web elements using JavaScript and create a responsive user experience.

## Features

- Dark Mode / Light Mode Toggle
- Real-Time Digital Clock
- Add New Tasks
- Delete Tasks
- Dynamic Task Counter
- Random Motivational Quotes
- Quote Fetching Using API
- Responsive Design
- Mobile-Friendly Layout

## Key Functionalities

### Theme Toggle

Users can switch between light and dark themes using a single button. Theme changes are applied dynamically using JavaScript and CSS class manipulation.

### Task Manager

Users can:

- Add new tasks
- Remove completed tasks
- View a live task count

All task elements are created dynamically through JavaScript.

### Live Clock

A digital clock updates every second using JavaScript timing functions.

### Quote Generator

Motivational quotes are fetched dynamically from:

https://dummyjson.com/quotes/random

The displayed quote updates whenever the user clicks the **New Quote** button.
